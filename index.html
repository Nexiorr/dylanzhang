À propos de moi

Actuellement en 2ème année de BUT Informatique à l’IUT de Lille, je suis passionné par le développement d'applications et l'informatique en général. Mon objectif est d'approfondir mes compétences dans le développement d’applications mobiles, web et d'apprendre à travailler efficacement en équipe.

En première année, j’ai participé à plusieurs projets de développement d'applications, notamment un assistant virtuel de voyage, où j'ai appris à gérer un projet de A à Z, en utilisant GitLab pour la gestion du code et du versioning.

Mon ambition est de décrocher un stage dans ce domaine afin d’acquérir plus d’expérience professionnelle et d’affiner mes compétences techniques.

MES PROJETS

FlyingDustries

FlyingDustries est une application codée principalement en java avec l'API javaFX pour gérer l'interface utilisateur / machine. Cette application utilise l'algorithme de Dijkstra pour calculer le chemin le plus court selon le choix de l'utilisateur.

Application de classification

L'objectif était de pouvoir créer une application qui permettrait de charger, classifier et afficher un ensemble de données. L'application permettrait d'importer des fichiers de types csv, ensuite l'affichage se fait depuis notre interface graphique sous un nuage de points et la classification se fait à l'aide d'un algorithme knn.

NerdFactory

NerdFactory est un projet de réalisation d'un jeu simple sur terminal réalisé sous 3 jours dans une équipe utilisant la méthodolie Agile.
